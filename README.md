# Teiguuu Dotfiles Collection

Репозиторий содержит конфигурационные файлы для различных дистрибутивов Linux.

## Структура проекта
* **/Arch dotfiles** — Текущая рабочая конфигурация (Arch Linux + Hyprland). Минимализм и системная эффективность.
* **/Debian dotfiles** — Заготовка под будущую конфигурацию на базе Debian.

## Описание
Проект переведен в формат коллекции для удобного управления несколькими окружениями. Подробности по установке и скриншоты находятся внутри соответствующих директорий.
